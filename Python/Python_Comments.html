Example 1 ‚Äì Single-Line & Inline Comment 

# This is a single-line comment
print("Hello!")  # This is an inline comment
  • # starts a single-line comment.
  • You can also put a comment after a line of code (inline).
  • Python ignores comments when running the program.
  • Output: Hello!
Example 2 ‚Äì Multi-Line Comment 

'''
This is a multi-line comment
It can be used to explain big blocks of code
'''

print("Python is fun!")
  • ''' (or """) can be used for multi-line comments.
  • Great for explaining full blocks of logic or code.
  • Python will skip anything inside the quotes.
  • Output: Python is fun!
Extra Example ‚Äì Inline Comment 

print("Sara loves drawing")  # This line prints a sentence about Sara
  • This is a single line with code and comment.
  • Comments after # are ignored by Python.
  • Only the text Sara loves drawing will be printed.
