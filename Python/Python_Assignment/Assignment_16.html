üìù Assignment 2: Multi-Line Docstrings

Your Task:

Create a function with a proper docstring explaining its purpose.

Steps:
  1. Write a function that converts Fahrenheit to Celsius
  2. Add a multi-line docstring immediately after the function definition
  3. Include: description, parameters, and return value in the docstring
  4. Call the function and print the result
Example Solution:
def fahrenheit_to_celsius(f_temp): """Convert Fahrenheit temperature to Celsius. Args: f_temp (float): Temperature in Fahrenheit Returns: float: Temperature in Celsius """ return (f_temp - 32) * 5/9 print(fahrenheit_to_celsius(75))
Docstrings are enclosed in triple quotes """ and describe what the function does.

üöß Assignment 3: Commenting Out Code

Your Task:

Practice commenting out code to disable execution without deleting.

Steps:
  1. Write a program with two different ways to calculate the average of three numbers
  2. Comment out the approach you're not using
  3. Add a comment explaining why you chose the other approach
  4. Include a TODO comment for future improvements
Example Solution:
num1, num2, num3 = 10, 20, 30 # Approach 1: Simple arithmetic average average = (num1 + num2 + num3) / 3 # Approach 2: Using sum() function (commented out) # average = sum([num1, num2, num3]) / 3 # Using Approach 1 for better readability with just 3 numbers print(f"Average: {average}") # TODO: Add input validation for non-numeric values
Commented code won't execute but remains visible for reference.

üìö Assignment 4: Module-Level Docstring

Your Task:

Create a Python file with a proper module docstring.

Steps:
  1. Create a new Python file (e.g., calculator.py)
  2. Add a module docstring at the very top explaining the file's purpose
  3. Include: module name, author, date, and description
  4. Add some functions with their own docstrings
Example Solution:
"""calculator.py - Basic math operations module Author: Jane Doe Date: 2023-10-15 This module provides basic arithmetic operations for educational purposes. """ def add(a, b): """Return the sum of two numbers.""" return a + b def multiply(a, b): """Return the product of two numbers.""" return a * b
Module docstrings should be the first thing in your Python file.

üè∑Ô∏è Assignment 5: Inline Comments

Your Task:

Write a program with effective inline comments explaining complex logic.

Steps:
  1. Create a program that checks if a number is prime
  2. Add inline comments explaining each step of the algorithm
  3. Make sure comments explain why, not just what the code does
  4. Include a function docstring
Example Solution:
def is_prime(n): """Check if a number is prime. Args: n (int): Number to check Returns: bool: True if prime, False otherwise """ if n <= 1: return False # Numbers less than 2 can't be prime # Check divisors from 2 to square root of n for i in range(2, int(n**0.5) + 1): if n % i == 0: return False # Found a divisor, not prime return True # No divisors found, it's prime
Good comments explain the reasoning behind the code.
