1 What is If Statement?

An if statement lets Python make decisions!

if condition:
    do something

If the condition is True, Python runs the indented code.

Important: Don't forget the colon : and indentation!
2 Simple If Example 
age = 18
if age >= 18:
    print("You are an adult.")
Output:
You are an adult.

If the condition is True, it will run the indented code.

3 If-Else Example 
age = 16
if age >= 18:
    print("Adult")
else:
    print("Not an adult")
Output:
Not an adult

If condition is False, Python runs the else block.

4 Input with If-Else 
age = int(input("Enter your age: "))
if age >= 18:
    print("Adult")
else:
    print("Minor")
Example Output:
Enter your age: 15
Minor

Combine input() with conditions to make interactive programs.

5 Common Mistake: = vs ==
Error Example: 
if x = 10:  # WRONG! This assigns 10 to x
    print("x is 10")
x = 10
if x == 10:  # CORRECT! This compares x to 10
    print("x is 10")
Output:
x is 10

= assigns a value, == compares values.

6 Using True in If 
if True:
    print("This always runs.")
Output:
This always runs.

You can use True directly (though not very useful in real code).

7 Comparison: 5 > 5 
if 5 > 5:
    print("Yes")
else:
    print("No")  # Runs "No"
Output:
No

5 is not greater than 5, so the condition is False.

8 String Comparison 
if "kareem" == "mango":
    print("Same")
else:
    print("Different")  # Runs "Different"
Output:
Different

Strings must match exactly (including case) to be equal.

9 Logical Operator: and 
if 5 > 5 and 5 < 6:
    print("Both True")
else:
    print("At least one is False")  # Runs here
Output:
At least one is False

and requires both conditions to be True.

10 Logical Operator: or 
if 5 > 5 or 5 < 6:
    print("At least one is True")  # Runs here
else:
    print("Both are False")
Output:
At least one is True

or only needs one condition to be True.

11 Using in, not in with If 
word = "kareem"
if "k" in word:
    print("Letter found")
if "z" not in word:
    print("Letter not found")
Output:
Letter found
Letter not found

Check if a value exists in a string (or list).

12 Using is, is not with If 
name = "Kareem"
nickname = "Kareem"
other = "Mango"
if name is nickname:
    print("Same object")
if name is not other:
    print("Different objects")
Output:
Same object
Different objects

is checks if two variables reference the exact same object.
