1 What is a Variable?

A variable is a name that stores a value. It's like a box to save data inside!

x = 5

Here, x stores the value 5.

2 Creating a Variable with a Number 
age = 12
print(age)
Output:
12

You can store numbers in variables for later use.

3 Creating a Variable with a String 
name = "Sara"
print(name)
Output:
Sara

Text (strings) go inside quotes when stored in variables.

4 Creating a Variable with a Boolean 
is_student = True
print(is_student)
Output:
True

Boolean variables store either True or False values.

5 Printing Text and Variable Together 
name = "Ali"
print("My name is", name)
Output:
My name is Ali

Use commas to combine text and variables in print().

6 Printing Multiple Variables in One Line 
name = "Mona"
age = 14
print("Name:", name, "Age:", age)
Output:
Name: Mona Age: 14

You can print several variables together in one statement.

7 Changing a Variable's Value 
x = 10
print(x)
x = 20
print(x)
Output:
10
20

Variables can change values - the new value replaces the old one!

8 Rules for Variable Names
  • Must start with a letter or underscore _
  • Cannot start with a number
  • No spaces allowed
  • Can only use letters, numbers, and underscores

Valid: my_age, _name, age2

Invalid: 2age, my age

9 Using type() with Variables 
name = "Sara"
print(type(name)) #
Output:
<class 'str'>

Check what type of data a variable holds with type().

10 Variables Help Make Code Reusable 
price = 100
print(price)
print(price + 50)
Output:
100
150

Store values once in variables, then use them many times!
