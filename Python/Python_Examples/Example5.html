1 Arithmetic Operators 
a = 10
b = 3
print(a + b) # 13
print(a - b) # 7
print(a * b) # 30
print(a / b) # 3.333...
print(a % b) # 1 (modulus - remainder after division)
Output:
13
7
30
3.3333333333333335
1
Operator Name Example
+ Addition a + b
- Subtraction a - b
* Multiplication a * b
/ Division a / b
% Modulus a % b
2 Assignment Operators 
x = 5
x += 3 # x = x + 3
print(x) # 8
x -= 2
print(x) # 6
x *= 2
print(x) # 12
x /= 3
print(x) # 4.0
x %= 3
print(x) # 1.0
Output:
8
6
12
4.0
1.0
Operator Example Same As
= x = 5 x = 5
+= x += 3 x = x + 3
-= x -= 2 x = x - 2
*= x *= 2 x = x * 2
/= x /= 3 x = x / 3
%= x %= 3 x = x % 3
3 Comparison Operators 
age = 15
print(age > 18) # False
print(age < 18) # True
print(age >= 15) # True
print(age <= 14) # False
print(age == 15) # True
print(age != 14) # True
Output:
False
True
True
False
True
True
Operator Name Example
== Equal a == b
!= Not equal a != b
> Greater than a > b
< Less than a < b
>= Greater than or equal a >= b
<= Less than or equal a <= b
4 Logical Operators 
a = 5
b = 10
print(a < 10 and b > 5) # True
print(a < 3 or b > 5) # True
print(not (a > b)) # True
Output:
True
True
True
Operator Description Example
and True if both are true x < 5 and x < 10
or True if at least one is true x < 5 or x < 4
not Reverse the result not(x < 5 and x < 10)
5 Identity Operators 
x = "Python"
y = "Python"
z = "Coding"
print(x is y) # True
print(x is not z) # True
Output:
True
True
Operator Description Example
is True if both variables are the same object x is y
is not True if both variables are not the same object x is not y
6 Membership Operators 
text = "Learning Python is fun"
print("Python" in text) # True
print("Coding" not in text) # True
Output:
True
True
Operator Description Example
in True if value is found in sequence x in y
not in True if value is not found in sequence x not in y
7 Input + Arithmetic Operator 
a = int(input("First number: "))
b = int(input("Second number: "))
print("Sum is", a + b)
Example Output:
First number: 5
Second number: 7
Sum is 12

Get numbers from user and perform arithmetic operations.

8 Input + Comparison Operator 
age = int(input("Enter your age: "))
print(age >= 18)
Example Output:
Enter your age: 15
False

Check if user's age meets certain conditions.

9 Logical Operator with Input 
x = int(input("Enter number: "))
print(x > 5 and x < 10)
Example Output:
Enter number: 7
True

Check if a number is between two values.

10 Membership Operator with Input 
sentence = input("Write a sentence: ")
print("Python" in sentence)
Example Output:
Write a sentence: I love Python
True

Check if certain words appear in user's input.
