1 What is a Data Type?

In Python, everything has a type. A data type tells Python what kind of value something is ‚Äî a number, text, or True/False!

2 Numbers - Integers and Floats 
print(5 + 3) # 8 (integer)
print(7.5 + 2.5) # 10.0 (float)
Output:
8
10.0

Numbers are used for math. Integers are whole numbers, floats have decimals.

3 Strings - Text 
print("5+3") # 5+3 (text, not math!)
print('Hello World')
Output:
5+3
Hello World

Anything inside quotes " " or ' ' is text (called a "string").

4 Booleans - True or False 
print(True)
print(False)
Output:
True
False

Booleans are special types that are only True or False ‚Äî perfect for yes/no questions!

5 Difference: Number vs. String 
print(5 + 5) # 10 (adds numbers)
print("5+5") # 5+5 (just prints text)
Output:
10
5+5

No quotes = math happens. Quotes = Python treats it as text.

6 Checking the Type 
print(type(5)) # 
print(type("Hello")) # 
print(type(True)) #
Output:
<class 'int'>
<class 'str'>
<class 'bool'>

Use type() to check what type something is.

7 Math with Numbers 
print(10 - 2)
print(4 * 5)
print(12 / 3)
Output:
8
20
4.0

Python can do all basic math operations when using numbers.

8 Text is Not Math 
print("10 - 2") # Python prints: 10 - 2
Output:
10 - 2

Text inside quotes is printed exactly as written - no calculations!

9 Booleans are Special 
is_raining = True
print(is_raining)
Output:
True

Booleans are used for decisions (True/False answers) in programs.

10 Mixing Types (Careful!) 
print(5 + "5") # This will give an error!
Note: This will cause a TypeError because you can't add a number and text directly!

Python needs values to be the same type for math operations.
