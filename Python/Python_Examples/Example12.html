1 What is a Dictionary?

A dictionary stores data in key-value pairs:

  • Written with {} curly braces
  • Keys must be unique and immutable (strings, numbers, tuples)
  • Values can be any data type
student = {
    "name": "Sara",
    "age": 15,
    "grade": "A"
}
print(student)
Output:
{'name': 'Sara', 'age': 15, 'grade': 'A'}
2 Access Dictionary Items 
# Method 1: Square brackets
print(student["name"]) # Sara

# Method 2: get() method
print(student.get("age")) # 15
Output:
Sara
15
Note: get() is safer as it returns None if key doesn't exist.
3 Change Dictionary Items 
student["age"] = 16
print(student)
Output:
{'name': 'Sara', 'age': 16, 'grade': 'A'}
4 Add Dictionary Items 
student["city"] = "Cairo"
print(student)
Output:
{'name': 'Sara', 'age': 16, 'grade': 'A', 'city': 'Cairo'}
5 Remove Dictionary Items

a. Using pop()

Remove item by key.

student.pop("grade")
print(student)
Output:
{'name': 'Sara', 'age': 16, 'city': 'Cairo'}

b. Using popitem()

Remove the last inserted item.

student.popitem()
print(student)
Output:
{'name': 'Sara', 'age': 16}

c. Using clear()

Empty the dictionary.

student.clear()
print(student)
Output:
{}
6 Copy Dictionaries

a. Using copy()

Create a copy of the dictionary.

student = {"name": "Sara", "age": 15}
new_student = student.copy()
new_student["age"] = 16
print("Original:", student)
print("Copy:", new_student)
Output:
Original: {'name': 'Sara', 'age': 15}
Copy: {'name': 'Sara', 'age': 16}

b. Using fromkeys()

Create dictionary from keys.

keys = ("name", "age", "grade")
new_student = dict.fromkeys(keys, "Unknown")
print(new_student)
Output:
{'name': 'Unknown', 'age': 'Unknown', 'grade': 'Unknown'}
7 Other Useful Methods 
student = {"name": "Sara", "age": 15, "grade": "A"}

# items()
print("Items:", student.items())

# keys()
print("Keys:", student.keys())

# values()
print("Values:", student.values())

# setdefault()
student.setdefault("gender", "Female")
print("After setdefault:", student)

# update()
student.update({"age": 17, "city": "Giza"})
print("After update:", student)
Output:
Items: dict_items([('name', 'Sara'), ('age', 15), ('grade', 'A')])
Keys: dict_keys(['name', 'age', 'grade'])
Values: dict_values(['Sara', 15, 'A'])
After setdefault: {'name': 'Sara', 'age': 15, 'grade': 'A', 'gender': 'Female'}
After update: {'name': 'Sara', 'age': 17, 'grade': 'A', 'gender': 'Female', 'city': 'Giza'}
8 Nested Dictionaries 
students = {
    "student1": {
        "name": "Ali",
        "age": 15
    },
    "student2": {
        "name": "Mona",
        "age": 14
    }
}

print(students["student1"]["name"]) # Ali
print(students["student2"]["age"]) # 14
Output:
Ali
14

Dictionaries can contain other dictionaries for complex data structures.

9 Dictionary Methods Cheat Sheet

clear() Removes all items dict.clear()
copy() Returns a copy new = dict.copy()
fromkeys() Creates dict from keys dict.fromkeys(keys, value)
get() Returns value for key dict.get(key)
items() Returns key-value pairs dict.items()
keys() Returns keys dict.keys()
pop() Removes item by key dict.pop(key)
popitem() Removes last inserted item dict.popitem()
setdefault() Sets default value for key dict.setdefault(key, value)
update() Updates dictionary dict.update(other_dict)
