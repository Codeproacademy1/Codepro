1 What is a For Loop?

A for loop repeats code for each item in a collection:

for item in collection:
    # do something with item
Key Points:
  • The loop runs once for each item in the collection
  • You can use any variable name instead of "item"
  • The code block must be indented
2 For Loop with List

Loop through each item in a list:

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)
Output:
apple
banana
cherry
3 For Loop with Tuple

Tuples work just like lists in for loops:

colors = ("red", "green", "blue")
for color in colors:
    print(color)
Output:
red
green
blue
4 For Loop with Set

Sets can be looped, but order is random:

animals = {"dog", "cat", "bird"}
for animal in animals:
    print(animal)
Possible Output:
cat
dog
bird
Note: Sets are unordered, so the loop order may vary each run.
5 For Loop with Dictionary

a. Loop through keys:

student = {"name": "Ali", "age": 15}
for key in student:
    print(key)
Output:
name
age

b. Loop through values:

for value in student.values():
    print(value)
Output:
Ali
15

c. Loop through both keys and values:

for key, value in student.items():
    print(key, value)
Output:
name Ali
age 15
6 For Loop with String

Loop through each character in a string:

for letter in "hello":
    print(letter)
Output:
h
e
l
l
o
7 For Loop with Range

Basic range:

for i in range(5):
    print(i)
Output:
0
1
2
3
4

Starting from 1:

for i in range(1, 6):
    print(i)
Output:
1
2
3
4
5
8 Nested For Loops

A loop inside another loop:

colors = ["red", "blue"]
fruits = ["apple", "banana"]
for color in colors:
    for fruit in fruits:
        print(color, fruit)
Output:
red apple
red banana
blue apple
blue banana
9 Break and Continue in Loops

Break: stop the loop early

for x in "banana":
    if x == "n":
        break
    print(x)
Output:
b
a

Continue: skip the current step

for x in "banana":
    if x == "n":
        continue
    print(x)
Output:
b
a
a
10 For Loop Cheat Sheet
Concept Syntax Description
Basic Loop for x in collection: Loops through each item
Range Loop for i in range(5): Loops 5 times (0-4)
Range with Start for i in range(1,6): Loops from 1-5
Dictionary Keys for key in dict: Loops through keys
Dictionary Values for val in dict.values(): Loops through values
Dictionary Items for k,v in dict.items(): Loops through both
Break break Exits the loop
Continue continue Skips current iteration
