1 What is a Set?

A set is a collection that:

  • Is unordered (no index)
  • Contains no duplicates
  • Uses {} curly braces (but empty sets use set())
fruits = {"apple", "banana", "cherry"}
print(fruits)
Possible Output:
{'banana', 'apple', 'cherry'}
Note: The order may change because sets are unordered!
2 Access Set Items 
for fruit in fruits:
    print(fruit)
Possible Output:
banana
apple
cherry
Remember: You cannot access set items by index like fruits[0]!
3 Add Set Items

a. Using add()

Add one item to the set.

fruits.add("orange")
print(fruits)
Possible Output:
{'banana', 'apple', 'cherry', 'orange'}

b. Using update()

Add multiple items from any iterable.

fruits.update(["mango", "grape"])
print(fruits)
Possible Output:
{'banana', 'apple', 'cherry', 'orange', 'mango', 'grape'}
4 Remove Set Items

a. Using remove()

Remove item by value (raises error if not found).

fruits.remove("banana")
print(fruits)
Possible Output:
{'apple', 'cherry', 'orange', 'mango', 'grape'}

b. Using discard()

Remove item safely (no error if missing).

fruits.discard("banana")  # No error even if "banana" not present
print(fruits)

c. Using pop()

Remove and return a random item.

item = fruits.pop()
print("Removed:", item)
print("Remaining:", fruits)
Possible Output:
Removed: apple
Remaining: {'cherry', 'orange', 'mango', 'grape'}

d. Using clear()

Remove all items.

fruits.clear()
print(fruits)
Output:
set()
5 Join Sets

a. Using union()

Combine sets and create new set.

set1 = {"a", "b", "c"}
set2 = {"d", "e", "f"}
set3 = set1.union(set2)
print(set3)
Possible Output:
{'a', 'b', 'c', 'd', 'e', 'f'}

b. Using update()

Add items from another set into current set.

set1.update(set2)
print(set1)
Possible Output:
{'a', 'b', 'c', 'd', 'e', 'f'}
6 Other Set Methods 
# Create example sets
a = {1, 2, 3}
b = {2, 3, 4}

# copy()
new_set = a.copy()
print("Copy:", new_set)

# difference()
print("a - b:", a.difference(b))  # {1}

# difference_update()
a.difference_update(b)
print("After difference_update:", a)  # {1}

# intersection()
print("Common items:", {1,2,3}.intersection({2,3,4}))  # {2, 3}

# intersection_update()
x = {1, 2, 3}
x.intersection_update({2, 3, 4})
print("After intersection_update:", x)  # {2, 3}

# symmetric_difference()
print("Unique items:", {1,2,3}.symmetric_difference({2,3,4}))  # {1, 4}

# symmetric_difference_update()
y = {1, 2, 3}
y.symmetric_difference_update({2, 3, 4})
print("After symmetric_difference_update:", y)  # {1, 4}

# isdisjoint()
print("Disjoint?:", {1,2}.isdisjoint({3,4}))  # True

# issubset()
print("Subset?:", {1,2}.issubset({1,2,3}))  # True

# issuperset()
print("Superset?:", {1,2,3}.issuperset({1,2}))  # True
