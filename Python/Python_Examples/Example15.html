1 What is a Function?

A function is a reusable block of code that performs a specific task:

def function_name():
    # do something

# Call the function
function_name()
Key Points:
  • Functions help organize code into logical chunks
  • They make code reusable and easier to maintain
  • The def keyword defines a function
  • Function names should be lowercase with underscores
2 Simple Function Example

A function that prints a greeting message:

def greet():
    print("Hello, welcome to Python!")

# Call the function
greet()
Output:
Hello, welcome to Python!
3 Function with Parameters

Functions can accept input values (parameters):

def greet(name):
    print("Hello", name)

# Call with argument
greet("Sara")
Output:
Hello Sara
Note: name is a parameter, "Sara" is an argument
4 Function Returning a Value

Functions can return results using the return statement:

def add(a, b):
    return a + b

result = add(5, 3)
print(result)
Output:
8
5 Using Data Types Inside Functions

Functions can work with any Python data type:

def check_type(x):
    print(type(x))

check_type(10)        # int
check_type(3.14)      # float
check_type("Hello")   # str
check_type(True)      # bool
Output:
<class 'int'>
<class 'float'>
<class 'str'>
<class 'bool'>
6 Using Variables Inside Functions

Functions can create and use their own variables:

def create_message():
    msg = "Python is fun!"
    print(msg)

create_message()
Output:
Python is fun!
Scope Note: Variables inside functions are local - they only exist inside the function.
7 Functions with Numbers and Operations

Functions can perform mathematical operations:

def multiply(x, y):
    return x * y

print(multiply(4, 5))
Output:
20
8 Functions with Conditions (if, else)

Functions can include conditional logic:

def check_age(age):
    if age >= 18:
        print("Adult")
    else:
        print("Minor")

check_age(17)
Output:
Minor
9 Functions with Match-Case

Python 3.10+ match-case statements in functions:

def day_type(day):
    match day:
        case "Saturday" | "Sunday":
            print("Weekend")
        case _:
            print("Weekday")

day_type("Sunday")
Output:
Weekend
10 Functions with Collections

a. Working with Lists:

def list_items(mylist):
    for item in mylist:
        print(item)

list_items(["apple", "banana", "cherry"])

b. Working with Tuples:

def show_tuple(mytuple):
    for item in mytuple:
        print(item)

show_tuple(("red", "green", "blue"))

c. Working with Sets:

def show_set(myset):
    for item in myset:
        print(item)

show_set({"dog", "cat", "bird"})

d. Working with Dictionaries:

def show_dict(mydict):
    for key, value in mydict.items():
        print(key, value)

show_dict({"name": "Ali", "age": 15})
11 Functions with Loops

a. For Loop Inside Function:

def count_to_five():
    for i in range(1, 6):
        print(i)

count_to_five()

b. While Loop Inside Function:

def countdown(n):
    while n > 0:
        print(n)
        n -= 1

countdown(5)
12 Function Cheat Sheet
Concept Syntax Description
Define Function def func(): Creates a new function
Parameters def func(param): Function inputs
Return Value return value Sends result back
Call Function func() Executes the function
Local Variables var = value Variables inside function
Conditionals if/else Decision making
Loops for/while Repeating code
Match-Case match case Pattern matching (Python 3.10+)
