üßë‚Äçüíª MicroPython Code: Control DC Motor

from machine import Pin
from time import sleep

in1 = Pin(15, Pin.OUT)
in2 = Pin(14, Pin.OUT)

def forward():
    in1.high()
    in2.low()

def reverse():
    in1.low()
    in2.high()

def stop():
    in1.low()
    in2.low()

while True:
    print("Forward")
    forward()
    sleep(2)
    
    print("Reverse")
    reverse()
    sleep(2)
    
    print("Stop")
    stop()
    sleep(2)

üîÅ How It Works:

Function Motor Behavior
IN1 = HIGH, IN2 = LOW Rotates forward
IN1 = LOW, IN2 = HIGH Rotates backward
Both LOW Stops motor

You can also use PWM for speed control.
