üîπ Part 3: Programming the LED with Arduino

üß† What We'll Do:

We'll write a simple program to blink the LED on and off every second.

üìÑ Code:

int ledPin = 13; // The pin where the LED is connected

void setup() {
  pinMode(ledPin, OUTPUT); // Set the pin as an OUTPUT
}

void loop() {
  digitalWrite(ledPin, HIGH); // Turn ON the LED
  delay(1000); // Wait for 1 second
  digitalWrite(ledPin, LOW); // Turn OFF the LED
  delay(1000); // Wait for 1 second
}

üí° What Happens:

  • pinMode(ledPin, OUTPUT): Prepares the pin to send voltage
  • digitalWrite(HIGH): Sends 5V, turning the LED ON
  • delay(1000): Waits 1000 milliseconds = 1 second
  • digitalWrite(LOW): Turns LED OFF
