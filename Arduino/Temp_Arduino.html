üîπ Part 3: Arduino Code ‚Äì Read & Display Temperature

üßæ Basic Code Using Serial Monitor:

int sensorPin = A0;
float voltage, temperature;

void setup() {
  Serial.begin(9600);
}

void loop() {
  int sensorValue = analogRead(sensorPin);

  voltage = sensorValue * (5.0 / 1023.0); // Convert ADC value to voltage
  temperature = voltage * 100; // LM35 gives 10mV per ¬∞C

  Serial.print("Temperature: ");
  Serial.print(temperature);
  Serial.println(" ¬∞C");

  delay(1000);
}

üîé Explanation:

  • analogRead() gives value from 0 to 1023 (10-bit)
  • Arduino's ADC range is 0‚Äì5V
  • Formula: voltage = analogRead √ó (5.0 / 1023.0)
  • Temperature conversion: temperature (¬∞C) = voltage √ó 100

üîÅ Alternatives: DHT11 or DHT22

If using DHT11 or DHT22, use the DHT library and follow this:

#include "DHT.h"

#define DHTPIN 2
#define DHTTYPE DHT11 // or DHT22

DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(9600);
  dht.begin();
}

void loop() {
  float temp = dht.readTemperature();

  Serial.print("Temp: ");
  Serial.print(temp);
  Serial.println(" C");
  delay(2000);
}

DHT11 also supports humidity reading.
