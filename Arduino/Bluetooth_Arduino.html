Feature Description
Type Classic Bluetooth (not BLE)
Mode Master or Slave (default is Slave)
Voltage 3.3V logic, but works with 5V on VCC

Pin Description

ðŸ”§ Voltage Divider for RXD:

Use two resistors to protect the HC-05 from 5V:

  • 1kÎ© between Arduino TX (pin 11) and HC-05 RX
  • 2kÎ© between HC-05 RX and GND

This creates ~3.3V from Arduino's 5V signal.

ðŸ”¹ Part 3: Arduino Code â€“ Control LED from Phone

#include <SoftwareSerial.h>

SoftwareSerial BTSerial(10, 11); // RX, TX

int ledPin = 13;
char incomingChar;

void setup() {
  pinMode(ledPin, OUTPUT);
  BTSerial.begin(9600);
  Serial.begin(9600);
}

void loop() {
  if (BTSerial.available()) {
    incomingChar = BTSerial.read();
    Serial.print("Received: ");
    Serial.println(incomingChar);

    if (incomingChar == '1') {
      digitalWrite(ledPin, HIGH);
    } else if (incomingChar == '0') {
      digitalWrite(ledPin, LOW);
    }
  }
}
