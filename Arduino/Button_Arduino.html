üîπ Part 1: What is a Button?

‚úÖ Definition:

A button is a type of switch that completes or breaks a circuit when pressed. It allows or stops the flow of current.

üì¶ Types of Buttons:

  • Normally Open (NO): Default state is open (no current). Closes when pressed.
  • Normally Closed (NC): Default state is closed (current flows). Opens when pressed.

In Arduino projects, we mostly use Normally Open (NO) buttons.

üîç How It Works:

When you press a button:

  1. The circuit closes.
  2. Current flows through.
  3. Arduino detects HIGH or LOW based on wiring.

When not pressed, the circuit stays open unless you use an internal pull-up or pull-down resistor.

üîπ Part 3: Arduino Code Examples

üßæ Basic Code: Turn ON LED when Button is Pressed

int buttonPin = 7; // Button connected to pin 7
int ledPin = 13; // LED connected to pin 13
int buttonState = 0; // Variable to store button state

void setup() {
  pinMode(buttonPin, INPUT); // Set pin 7 as input
  pinMode(ledPin, OUTPUT); // Set pin 13 as output
}

void loop() {
  buttonState = digitalRead(buttonPin); // Read button state

  if (buttonState == HIGH) {
    digitalWrite(ledPin, HIGH); // Turn ON LED
  } else {
    digitalWrite(ledPin, LOW); // Turn OFF LED
  }
}

‚ö†Ô∏è Note:

This basic circuit requires an external 10kŒ© pull-down resistor to work properly. Without it, the input might float between HIGH and LOW unpredictably.

üîÑ Alternative: Using INPUT_PULLUP (No External Resistor)

üí° Internal Pull-Up:

If you don't want to use a 10kŒ© resistor, Arduino has built-in pull-up resistors that can be enabled with INPUT_PULLUP.

üßæ Updated Code:

void setup() {
  pinMode(buttonPin, INPUT_PULLUP); // Enable internal pull-up
  pinMode(ledPin, OUTPUT);
}

void loop() {
  buttonState = digitalRead(buttonPin);

  if (buttonState == LOW) { // Button pressed (pull-up logic)
    digitalWrite(ledPin, HIGH);
  } else {
    digitalWrite(ledPin, LOW);
  }
}

Important Note on INPUT_PULLUP:

With INPUT_PULLUP, the logic is reversed:

  • LOW = button pressed
  • HIGH = button not pressed

This is because the internal pull-up resistor keeps the pin HIGH when the button isn't pressed, and pressing the button connects it to GND (LOW).
