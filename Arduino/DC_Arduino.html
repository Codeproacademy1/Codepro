üîπ Part 3: Arduino Code ‚Äì Turn Motor ON and OFF

üßæ Code Example (ON/OFF control):

int motorPin = 9; // Connected to transistor base via resistor

void setup() {
  pinMode(motorPin, OUTPUT);
}

void loop() {
  digitalWrite(motorPin, HIGH); // Motor ON
  delay(2000); // Wait 2 seconds
  digitalWrite(motorPin, LOW); // Motor OFF
  delay(2000); // Wait 2 seconds
}

üî∏ Part 4: Speed Control Using PWM

You can control the motor speed using analogWrite().

üßæ PWM Speed Control Code:

int motorPin = 9;

void setup() {
  pinMode(motorPin, OUTPUT);
}

void loop() {
  for (int speed = 0; speed <= 255; speed += 5) {
    analogWrite(motorPin, speed); // Gradually increase speed
    delay(50);
  }
  for (int speed = 255; speed >= 0; speed -= 5) {
    analogWrite(motorPin, speed); // Gradually decrease speed
    delay(50);
  }
}

üîπ Part 5: Optional ‚Äì Using L298N Motor Driver

The L298N is a motor driver module that allows:

  • Forward/backward control
  • Dual motor control
  • Speed + direction control

üßæ Wiring:

IN1 ‚Üí Arduino pin 8

IN2 ‚Üí Arduino pin 9

ENA ‚Üí Arduino pin 10 (for PWM speed)

+12V ‚Üí External motor power

GND ‚Üí Shared with Arduino

üßæ L298N Code:

int IN1 = 8;
int IN2 = 9;
int ENA = 10;

void setup() {
  pinMode(IN1, OUTPUT);
  pinMode(IN2, OUTPUT);
  pinMode(ENA, OUTPUT);
}

void loop() {
  digitalWrite(IN1, HIGH); // Motor forward
  digitalWrite(IN2, LOW);
  analogWrite(ENA, 150); // Speed control (0-255)
  delay(3000);

  digitalWrite(IN1, LOW); // Motor backward
  digitalWrite(IN2, HIGH);
  analogWrite(ENA, 150);
  delay(3000);

  digitalWrite(IN1, LOW); // Stop
  digitalWrite(IN2, LOW);
  delay(2000);
}
