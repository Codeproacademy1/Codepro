üîπ Part 1: What is a Potentiometer?

‚úÖ Definition:

A potentiometer (often called a pot) is a variable resistor. You can turn its knob to adjust resistance, which changes the voltage output.

With Arduino, it's commonly used to control brightness, speed, or volume by reading analog input.

üîç Structure:

A typical rotary potentiometer has 3 pins:

  • Left Pin (GND) ‚Äì connect to ground
  • Right Pin (VCC) ‚Äì connect to 5V
  • Middle Pin (wiper) ‚Äì sends a variable voltage (connect to A0)

üîπ Part 3: Arduino Code ‚Äì Read and Display Value

üßæ Basic Code Example:

int potPin = A0; // Middle pin connected to A0
int potValue = 0; // To store analog value

void setup() {
  Serial.begin(9600); // Start Serial Monitor
}

void loop() {
  potValue = analogRead(potPin); // Read value (0‚Äì1023)
  Serial.print("Potentiometer Value: ");
  Serial.println(potValue);
  delay(200);
}

üî¢ analogRead() Output:

  • The function reads a value from 0 to 1023
  • 0 = 0V
  • 1023 = 5V
  • The voltage changes as you turn the knob

üí° Example Use: Control LED Brightness

int potPin = A0;
int ledPin = 9;

void setup() {
  pinMode(ledPin, OUTPUT);
}

void loop() {
  int value = analogRead(potPin); // 0‚Äì1023
  int brightness = map(value, 0, 1023, 0, 255); // scale to 0‚Äì255
  analogWrite(ledPin, brightness); // adjust brightness
}
