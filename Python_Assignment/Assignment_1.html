Watch this video first to understand the basics of the print() function in Python.

1 Printing Simple Text with Double Quotes 
print("Hello, world!")
Output: Hello, world!

You can use double quotes " to print a text message.

2 Printing Simple Text with Single Quotes 
print('Python is fun!')
Output: Python is fun!

You can also use single quotes ' to print text.

3 Printing a Sentence Using Triple Double Quotes 
print("""Welcome to Python!
Let's learn together.""")
Output:
Welcome to Python!
Let's learn together.

Triple double quotes allow you to print sentences on multiple lines.

4 Printing a Sentence Using Triple Single Quotes 
print('''This is Python class.
Have fun coding!''')
Output:
This is Python class.
Have fun coding!

Triple single quotes also let you print multiple lines.

5 Printing a Number (Not Text!) 
print(5 + 5)
Output: 10

If you type a number without quotes, Python will treat it like math!

6 Printing a Text that Looks Like Math 
print("5 + 5")
Output: 5 + 5

If you use quotes around numbers and symbols, Python prints it as text!

7 Printing Two Messages in One Line 
print("Hello", "World")
Output: Hello World

You can print multiple things together by separating them with commas.

8 Printing Numbers and Text Together 
print("I am", 12, "years old")
Output: I am 12 years old

Mix text and numbers in the same print() by using commas.

9 Printing Emojis 
print("Python is awesome! üêç")
Output: Python is awesome! üêç

You can print emojis by putting them inside quotes.

 ^_^

You can create simple shapes by printing text art.
