üîÑ Assignment 2: Update Trick

Your Task:

Modify a tuple by converting it to a list and back.

Steps:
  1. Start with your fruit tuple from Assignment 1
  2. Convert it to a list using list()
  3. Change the third item in the list
  4. Convert back to a tuple using tuple()
  5. Print the modified tuple
Example Transformation:
Original: ('apple', 'banana', 'orange', 'grape', 'mango') Modified: ('apple', 'banana', 'pear', 'grape', 'mango')
This is how we "modify" tuples since they're immutable.
