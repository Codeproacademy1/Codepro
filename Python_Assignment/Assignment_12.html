Your Task:

Steps:
  1. Start with your student dictionary

Your Task:

Check for a key and add it with a default value if missing.

Steps:
  1. Start with your student dictionary
  2. Check if 'gender' key exists using in

Your Task:

Create a dictionary from a list with default values.

Steps:
  1. Create a list of subjects (e.g., ['Math', 'Science', 'History'])
