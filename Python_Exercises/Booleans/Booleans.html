Python Booleans - Kid-Friendly Questions

  1. 1. What is a Boolean?
    Hint: In Python, a Boolean is a type of value that can either be True or False.
  2. 2. Which words in Python represent Boolean values?
    Hint: In Python, Booleans start with capital letters.
  3. 3. If you write 5 > 3 in Python, what Boolean value will you get?
    Hint: Compare 5 and 3: is 5 greater than 3 or not?
  4. 4. If you write 10 < 5 in Python, will it return True or False?
    Hint: Compare the numbers carefully.
  5. 5. What does the == operator check in Python?
    Hint: It compares two values to see if they are the same.
  6. 6. If my_number = 7, what will Python output for my_number == 7?
    Hint: Think about whether 7 is equal to 7.
  7. 7. What does the != operator do?
    Hint: It checks if two values are not equal.
  8. 8. Which logical operator would make the expression True and False return False?
    Hint: Remember the three logical operators in Python: and, or, not.
  9. 9. How does the not operator affect a Boolean value?
    Hint: It flips the value from True to False or from False to True.
  10. 10. What is the result of True or False in Python?
    Hint: If at least one value is True, the whole thing becomes True.

Tips for Kids

Try in a Python Shell: Open a Python interpreter and type each question’s code or expression to see what happens.

Experiment: Change numbers in comparisons (for example, 5 > 3 to 2 > 10) and see how the results change.

Use Print Statements:
For instance:
print(5 > 3)
print(5 < 3)
You’ll see the Boolean values immediately in the output!
