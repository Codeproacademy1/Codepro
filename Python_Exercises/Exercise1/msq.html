Multiple Choice Question

Which of the following correctly prints "Hello, World!" in Python?

A) print Hello, World!
B) print("Hello, World!")
C) display("Hello, World!")
D) echo "Hello, World!"

Correct Answer:

B) print("Hello, World!")

In Python, print() is a function that requires parentheses.

Multiple Choice Question

What is the main purpose of the print() function?

A) To create a new variable
B) To get input from the user
C) To show output on the screen
D) To close the program

Correct Answer:

C) To show output on the screen

The print() function displays output to the console.

Multiple Choice Question

Which of the following will cause a syntax error?

A) print("Age:", 10)
B) print("Age: " + str(10))
C) print("Age: " 10)
D) print("Age:", str(10))

Correct Answer:

C) print("Age: " 10)

Missing comma or concatenation operator between string and number.

Multiple Choice Question

What is the default ending character of the print() function?

A) !
B) .
C) A new line (\n)
D) A space

Correct Answer:

C) A new line (\n)

By default, print() ends with a newline character.

Multiple Choice Question

Which escape character is used to create a new line in a string?

A) \t
B) \n
C) \\
D) \"

Correct Answer:

B) \n

The \n escape sequence creates a new line.

Multiple Choice Question

How can you print without moving to a new line?

A) print("Hello", skipline=False)
B) print("Hello", end=" ")
C) print("Hello", close="no")
D) print("Hello", stop="\n")

Correct Answer:

B) print("Hello", end=" ")

The end parameter controls what's printed at the end.

Multiple Choice Question

What is the output of print("3" + "4")?

A) 7
B) 34
C) 3 4
D) Error

Correct Answer:

B) 34

The + operator concatenates strings.

Multiple Choice Question

Which of the following will print two values with a space between them?

A) print("Hello" + "World")
B) print("Hello", "World")
C) print("Hello" "World")
D) print("Hello".concat("World"))

Correct Answer:

B) print("Hello", "World")

The comma in print() adds a space between arguments.

Multiple Choice Question

Which of the following prints:
Line1
Line2

A) print("Line1 Line2")
B) print("Line1\nLine2")
C) print("Line1/tLine2")
