True/False Question

The print() function in Python always adds a space between multiple arguments.

Correct Answer:

True

print() adds a space between arguments by default.

True/False Question

print("Python", 3.9) will print: Python3.9

Correct Answer:

False

It will print: Python 3.9 (with a space)

True/False Question

print("5" + 2) will cause an error.

Correct Answer:

True

You can't concatenate a string and integer directly.

True/False Question

print("Hello", end="*") will end the line with a star (*).

Correct Answer:

True

The end parameter specifies what to print at the end.

True/False Question

You can use print() to read data from the user.

Correct Answer:

False

print() is for output, input() is for reading user input.
