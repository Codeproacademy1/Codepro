1 What is a Tuple?

A tuple is like a list, but it cannot be changed (immutable).

  • Uses ( ) parentheses
  • Items are ordered
  • Items cannot be added, removed, or changed after creation
fruits = ("apple", "banana", "cherry")
print(fruits)
Output:
('apple', 'banana', 'cherry')
2 Access Tuple Items 
print(fruits[0]) # apple
print(fruits[2]) # cherry
Output:
apple
cherry
Note: Like lists, tuples use zero-based indexing.
3 Update Tuples (Workaround)

Tuples are immutable, but you can convert to list, change, then back to tuple.

x = ("apple", "banana", "cherry")
y = list(x)
y[1] = "orange"
x = tuple(y)
print(x)
Output:
('apple', 'orange', 'cherry')
Remember: This creates a new tuple, it doesn't modify the original.
4 Unpack Tuples

Assign tuple items to variables easily.

fruits = ("apple", "banana", "cherry")
(a, b, c) = fruits
print(a)
print(b)
print(c)
Output:
apple
banana
cherry
5 Join Tuples 
tuple1 = ("a", "b", "c")
tuple2 = ("d", "e", "f")
tuple3 = tuple1 + tuple2
print(tuple3)
Output:
('a', 'b', 'c', 'd', 'e', 'f')
6 Tuple Methods

a. count()

Counts how many times a value appears.

numbers = (1, 2, 2, 3, 2)
print(numbers.count(2)) # 3
Output:
3

b. index()

Finds the index (position) of a value.

fruits = ("apple", "banana", "cherry")
print(fruits.index("banana")) # 1
Output:
1
7 One Item Tuple (Important!) 
single = ("apple",) # Correct tuple
not_single = ("apple") # This is just a string

print(type(single))
print(type(not_single))
Output:
<class 'tuple'>
<class 'str'>
Important: For single-item tuples, you must include a comma!
8 Tuple with Different Types 
mix = ("apple", 3, True, 2.5)
print(mix)
Output:
('apple', 3, True, 2.5)

Tuples can contain any mix of data types.

Feature Tuple List
