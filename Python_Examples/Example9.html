1 What is a List?

A list is a collection of items inside square brackets [].

fruits = ["apple", "banana", "cherry"]
print(fruits)
Output:
['apple', 'banana', 'cherry']
Note: Lists can contain any data type (strings, numbers, even other lists).
2 Access List Items 
print(fruits[0]) # apple
print(fruits[2]) # cherry
Output:
apple
cherry
Remember: Python uses zero-based indexing (first item is index 0)!
3 Change List Items 
fruits[1] = "blueberry"
print(fruits)
Output:
['apple', 'blueberry', 'cherry']

Lists are mutable - you can change their contents after creation.

4 Add List Items

a. Using append()

Adds at the end of the list.

fruits.append("mango")
print(fruits)
Output:
['apple', 'blueberry', 'cherry', 'mango']

b. Using insert()

Add at a specific position.

fruits.insert(1, "orange")
print(fruits)
Output:
['apple', 'orange', 'blueberry', 'cherry', 'mango']

c. Using extend()

Add multiple items.

more_fruits = ["pear", "grape"]
fruits.extend(more_fruits)
print(fruits)
Output:
['apple', 'orange', 'blueberry', 'cherry', 'mango', 'pear', 'grape']
5 Remove List Items

a. Using remove()

Remove by value.

fruits.remove("banana")
print(fruits)
Output:
['apple', 'orange', 'blueberry', 'cherry', 'mango', 'pear', 'grape']
Note: Will raise an error if item not found!

b. Using pop()

Remove by index (or last item if no index).

fruits.pop(2) # removes third item
print(fruits)
fruits.pop() # removes last item
print(fruits)
Output:
['apple', 'orange', 'cherry', 'mango', 'pear', 'grape']
['apple', 'orange', 'cherry', 'mango', 'pear']

c. Using clear()

Removes all items.

fruits.clear()
print(fruits)
Output:
[]
6 Sort Lists

a. Using sort()

Sort in ascending order.

numbers = [3, 1, 4, 1, 5, 9, 2]
numbers.sort()
print(numbers)
Output:
[1, 1, 2, 3, 4, 5, 9]

b. Using reverse()

Reverse the list.

numbers.reverse()
print(numbers)
Output:
[9, 5, 4, 3, 2, 1, 1]
7 Copy Lists

a. Using copy()

Create a copy of the list.

original = [1, 2, 3]
new_list = original.copy()
new_list.append(4)
print("Original:", original)
print("New:", new_list)
Output:
Original: [1, 2, 3]
New: [1, 2, 3, 4]
Important: Without copy(), changes to one list would affect both!
8 Join Lists 
list1 = ["a", "b", "c"]
list2 = ["d", "e", "f"]

# Method 1: + operator
joined = list1 + list2
print(joined)

# Method 2: extend()
list1.extend(list2)
print(list1)
Output:
['a', 'b', 'c', 'd', 'e', 'f']
['a', 'b', 'c', 'd', 'e', 'f']
9 More List Methods

a. count()

Counts how many times an item appears.

numbers = [1, 2, 2, 3, 2]
print(numbers.count(2)) # 3
Output:
3

b. index()

Find the position of an item.

print(numbers.index(3)) # 3
Output:
3
