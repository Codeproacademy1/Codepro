1 What Are Numbers in Python?

Numbers are used for math, counting, measuring, and calculations in Python.

2 Three Main Types of Numbers

Python has 3 main types of numbers:

  • int Whole numbers (like 5, -3, 100)
  • float Decimal numbers (like 3.14, -0.5)
  • complex Special numbers with "real" and "imaginary" parts (for now, just know they exist)
a = 5 # int
b = 3.14 # float
c = 2 + 3j # complex
3 Checking the Type of a Number 
print(type(5)) # <class 'int'>
print(type(3.14)) # <class 'float'>
print(type(2 + 3j)) # <class 'complex'>
Output:
<class 'int'>
<class 'float'>
<class 'complex'>
4 Basic Math Operations 
print(5 + 3) # 8
print(10 - 4) # 6
print(2 * 6) # 12
print(9 / 3) # 3.0
print(10 % 3) # 1 (remainder)
Output:
8
6
12
3.0
1
5 Float Division 
print(7 / 2) # 3.5
Output:
3.5
Note: Division always returns a float, even if the result is a whole number!
6 Casting Numbers 
x = 5
y = float(x) # 5.0
z = int(3.99) # 3 (truncates, doesn't round)
Important: Converting float to int cuts off the decimal (doesn't round)
7 Doing Math with int and float 
a = 5
b = 2.5
print(a + b) # 7.5
Output:
7.5

When mixing int and float, Python automatically converts to float.

8 Working with Input and Numbers 
age = int(input("Enter your age: "))
print(age + 5)
Example Output:
Enter your age: 15
20
Remember: input() gives a string. You must cast to int or float for math!
9 Power Operator ** 
print(2 ** 3) # 8 (2*2*2)
print(10 ** 2) # 100
Output:
8
100

** means "raise to the power".

10 Floor Division // 
print(7 // 2) # 3
print(10.5 // 3) # 3.0
Output:
3
3.0

// gives whole number division (rounds down).

B1 Absolute Value 
print(abs(-7)) # 7
print(abs(3.5)) # 3.5
print(abs(-2.9)) # 2.9
Output:
7
3.5
2.9

abs() gives the positive version of a number.

B2 Rounding Numbers 
print(round(3.7)) # 4
print(round(3.2)) # 3
print(round(3.14159, 2)) # 3.14
Output:
4
3
3.14

round() makes numbers simpler. Add a second number for decimal places.
