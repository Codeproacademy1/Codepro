1 What is a While Loop?

A while loop repeats code as long as a condition is True:

while condition:
    # do something
Key Points:
  • The loop continues until the condition becomes False
  • You must update variables in the condition to avoid infinite loops
  • The code block must be indented
2 Simple While Loop Example

Print numbers 0 to 4:

i = 0
while i < 5:
    print(i)
    i += 1  # Don't forget to increment!
Output:
0
1
2
3
4
Warning: Forgetting to increment i would create an infinite loop!
3 While with Break

Stop the loop early using break:

i = 1
while i < 6:
    print(i)
    if i == 3:
        break
    i += 1
Output:
1
2
3
4 While with Continue

Skip part of the loop using continue:

i = 0
while i < 5:
    i += 1
    if i == 3:
        continue
    print(i)
Output:
1
2
4
5
Note: Notice how 3 is skipped because we used continue
5 While with Else

The else block runs after the while loop finishes normally (no break):

i = 0
while i < 3:
    print(i)
    i += 1
else:
    print("Finished!")
Output:
0
1
2
Finished!
6 Infinite While Loop

A loop that runs forever (until manually stopped or with break):

while True:
    print("This will never stop unless break is used!")
    break  # Remove this break to see real infinite loop
Output:
This will never stop unless break is used!
Warning: Be careful with infinite loops! Always test with a break condition first.
7 While Loop Input Example

Keep asking for a password until correct:

password = ""
while password != "1234":
    password = input("Enter password: ")
print("Access granted")
Example Session:
Enter password: abc
Enter password: test
Enter password: 1234
Access granted
8 While Loop Countdown

Print numbers backward (5 to 1):

i = 5
while i > 0:
    print(i)
    i -= 1
Output:
5
4
3
2
1
9 Common While Loop Mistakes

1. Forgetting to update the condition variable:

# Infinite loop!
i = 0
while i < 5:
    print(i)
    # Forgot i += 1

2. Wrong condition (loop never runs):

# Loop never starts
i = 10
while i < 5:
    print(i)
    i += 1

3. Missing indentation:

# Syntax error!
i = 0
while i < 5:
print(i)  # Missing indentation
i += 1
Tip: Always test your while loops with simple values first to avoid these common mistakes!
