Video Tutorial

Watch this video to learn how to get user input and convert between data types.

1 Getting Input from User 
name = input()
print(name)
Example Output:
(Whatever the user types)

input() pauses the program and waits for user to type something.

2 Input with a Question 
name = input("What is your name? ")
print(name)
Example Output:
What is your name? Alex
Alex

Add a prompt inside input() to tell users what to enter.

3 Storing Input in a Variable 
city = input("Where do you live? ")
print(city)
Example Output:
Where do you live? New York
New York

The input is stored in the variable and can be used later.

4 Input is Always a String 
age = input("Enter your age: ")
print(age) # It's a string!
print(type(age))
Example Output:
Enter your age: 12
12
<class 'str'>
Important: Even numbers entered become strings!
5 Casting Input to Integer 
age = int(input("Enter your age: "))
print(age)
print(type(age))
Example Output:
Enter your age: 12
12
<class 'int'>

Use int() to convert input to a whole number.

6 Casting Input to Float 
price = float(input("Enter the price: "))
print(price)
print(type(price))
Example Output:
Enter the price: 9.99
9.99
<class 'float'>

Use float() for decimal numbers.

7 Casting to String 
year = str(2025)
print(year)
print(type(year))
Output:
2025
<class 'str'>

Sometimes you need to convert numbers back to strings.

8 Casting a Saved Variable 
value = "100"
value = int(value)
print(value)
print(type(value))
Output:
100
<class 'int'>

You can change a variable's type after it's created.

9 Adding Two Inputs After Casting 
num1 = int(input("Enter first number: "))
num2 = int(input("Enter second number: "))
print(num1 + num2)
Example Output:
Enter first number: 5
Enter second number: 7
12

Cast inputs to numbers before doing math operations.

10 Important: Cast Before Using 
a = input("Enter a number: ")
b = input("Enter another number: ")
print(a + b) # This will join text, not add numbers!

# Correct way:
a = int(a)
b = int(b)
print(a + b) # Now it adds numbers
Example Output (Wrong Way):
Enter a number: 5
Enter another number: 7
57

Example Output (Correct Way):
12
Warning: Always cast to int/float before math operations!
