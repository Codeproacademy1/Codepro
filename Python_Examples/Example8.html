1 What is Match Statement?

The match statement is like a super-powered if-elif-else. It checks a value against many possible patterns.

Note: Match statements require Python 3.10 or newer.
2 Simple Match Example 
color = "red"
match color:
    case "red":
        print("Color is Red!")
    case "blue":
        print("Color is Blue!")
    case _:
        print("Unknown Color!")
Output:
Color is Red!

Python checks the value of color and runs the matching case.

3 Match with Input 
day = input("Enter a day: ")
match day:
    case "Monday":
        print("Start of the week!")
    case "Friday":
        print("Almost weekend!")
    case _:
        print("Just another day")
Example Output:
Enter a day: Friday
Almost weekend!
4 Match with Numbers 
num = int(input("Enter a number: "))
match num:
    case 1:
        print("One")
    case 2:
        print("Two")
    case _:
        print("Something else")
Example Output:
Enter a number: 2
Two
5 Match with Casting 
age = int(input("Enter your age: "))
match age:
    case 18:
        print("You are 18!")
    case 21:
        print("You are 21!")
    case _:
        print("Another age")
Example Output:
Enter your age: 21
You are 21!
Remember: Always cast input if you're expecting numbers!
6 Match and Operators (Inside Cases) 
score = int(input("Enter your score: "))
match score:
    case _ if score >= 90:
        print("Excellent!")
    case _ if score >= 75:
        print("Good!")
    case _:
        print("Keep trying!")
Example Output:
Enter your score: 85
Good!

Use case _ if condition: to add conditions inside cases.

7 Default Case: _

The case _: is the "catch-all" that runs when nothing else matches.

fruit = "banana"
match fruit:
    case "apple":
        print("It's an apple")
    case "orange":
        print("It's an orange")
    case _:
        print("Unknown fruit")
Output:
Unknown fruit
8 Common Mistake: Forgetting match or case
Wrong: 
switch value:  # Python doesn't have "switch"
    case "something":
        print("done")
value = "hello"
match value:  # Correct syntax
    case "hello":
        print("done")
Output:
done
9 Match with Strings and Membership 
text = input("Type something: ")
match text:
    case _ if "hello" in text:
        print("You said hello!")
    case _:
        print("No hello found.")
Example Output:
Type something: hello world
You said hello!
10 Match with Identity Operators 
a = "Python"
b = "Python"
c = "Java"

match a:
    case _ if a is b:
        print("Same object as b")
    case _ if a is not c:
        print("Different object from c")
Output:
Same object as b

While not common, you can use is and is not in match guards.
